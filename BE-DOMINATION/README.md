# BE-DOMINATION

A compact learning repository for backend and JavaScript fundamentals. It contains examples of core JavaScript (Fundamentals), asynchronous patterns and modern features (Advanced), and a minimal Node.js example (Node.js).

## Project Overview

- **Purpose:** Educational examples and exercises covering JavaScript basics, async patterns (callbacks, promises, async/await), module usage, and a tiny Node.js app.
- **Audience:** Beginners to intermediate JavaScript learners who want runnable examples and small reference files.

## Repository Structure

- `Fundamentals/` — basic HTML and `script.js` examples demonstrating core language features.
- `Advanced/` — files: `async.js`, `callbacks.js`, `promises.js`, `importExport.js`, `script.js` showing modern and asynchronous patterns.
- `Node.js/` — minimal Node project with `package.json` and `index.js` (prints a greeting).
- `Understanding Internet/` — notes and reading material.

## Quick Start

Prerequisites: Node.js (v16+ recommended) and `npm`.

Run the Node example:

```bash
cd Node.js
npm install
node index.js
```

Open the fundamentals examples:

- Open `Fundamentals/index.html` in your browser to run the example page.
