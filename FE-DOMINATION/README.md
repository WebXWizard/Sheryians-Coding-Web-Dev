# Frontend Domination

Frontend Domination is a hands-on learning repository for practicing core frontend development concepts. The project is organized module by module and covers the journey from basic setup and HTML fundamentals to CSS layouts, responsive design, animations, JavaScript basics, and advanced JavaScript concepts.

## Project Description

This repository contains course notes, practice files, and mini frontend projects created while learning web development. It focuses on building a strong foundation in HTML, CSS, responsive layouts, UI styling, and JavaScript programming through small examples and project-based exercises.

The repo is useful for revising frontend concepts, experimenting with code, and tracking progress through a structured frontend development curriculum.

## What Is Included

- Getting started notes and weekly learning structure
- HTML practice files for headings, paragraphs, formatting, lists, forms, semantic tags, media, links, divs, IDs, and classes
- CSS basics with styling examples
- CSS Grid layout practice
- CSS mini projects such as CottonWeave, ShopJacket, and Trendline
- Responsive web development practice
- CSS animation examples
- JavaScript fundamentals including variables, data types, conditions, loops, arrays, objects, and questions
- Asynchronous JavaScript practice
- Advanced JavaScript examples covering `this`, constructor functions, event listeners, `call`, `apply`, and `bind`

## How To Run

This is a static frontend practice repository. No package installation is required.

1. Clone or download the repository.
2. Open any `.html` file directly in your browser.
3. To edit the code, open the folder in VS Code or another code editor.
4. For the best development experience, use the Live Server extension in VS Code.

Example files to open:

```text
Module - 03 - Everything About CSS/7 - CSS Projects/Trendline/index.html
Module - 03 - Everything About CSS/7 - CSS Projects/ShopJacket/index.html
Module - 03 - Everything About CSS/7 - CSS Projects/CottonWeave/index.html
Module - 04 - Everything About JavaScript/13 - Advance JavaScript - God level JS/index.html
```

## Learning Goal

The goal of this repository is to build frontend development confidence by practicing concepts step by step. Each module adds a new layer of understanding, starting from basic HTML structure and moving toward styling, layouts, responsiveness, animations, and JavaScript interactivity.
